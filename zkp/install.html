How to Install

Step by step guide to install mods.

Download the Engine

All my charts and mods are built for P-Slice Engine (Psych Engine)

> Download P-Slice Engine

📱 Download the P-Slice Engine APK and install it. Then open the app once and grant it permission to manage files — this creates the mods folder automatically.

> Download P-Slice APK

Download the Mod

Go to the Mods section, pick the one you like and download it. It will come as a .zip or .rar file.

📥 Go to the Mods section and download the mod. It saves as a .zip or .rar in your Downloads folder.

Extract to the 'mods' Folder

Unzip the file you downloaded. Move that folder inside the mods/ directory of P-Slice Engine:

C:\P_Slice_Engine\mods\Your_Mod_Here\

📂 Open ZArchiver and enable hidden folders first: ☰ Menu → Settings → Show hidden files. Then find the downloaded file, extract it and move the folder to:

.P-SliceEngine/mods/

📁 This folder appears next to Music, Movies and Downloads once you open the app at least once.

Play!

Open the game .exe. Go to Freeplay and you will see the songs.

🎮 Open P-Slice Engine and go to Freeplay — the mod songs will be there. Time to destroy your screen! 📱💥
