How to Install

Step by step guide to install mods.

Download the Engine

All my charts and mods are built for P-Slice Engine (Psych Engine). It is the most stable and standard engine right now.

> Download P-Slice Engine

Download the Mod

Go to the Mods section, pick the one you like and download it. It will come as a .zip or .rar file.

Extract to the 'mods' Folder

Unzip the file you downloaded. You will get a folder with the mod name. Move that entire folder inside the mods/ directory of P-Slice Engine:

C:\P_Slice_Engine\mods\Your_Mod_Here\

Play!

Open the game .exe. Go to the Freeplay menu and you will see the songs ready to go. Time to destroy your keyboard!
